PROJETO AGENDA DE DEVEDORES

O projeto consiste na criação de uma agenda de devedores, formulada utilizando PHP e JavaScript, assim como CSS e HTML para a estruturação do site, onde o usuário pode adicionar um novo devedor, assim como consultar os devedores já listados, que ficam armazenados em um banco de dados, podendo realizar ações de alterar status do pagamento, assim como excluir algum dos devedores.
